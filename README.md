# dwm Keybinds
**Mod1** is binded to **Alt**.__
**Mod1 + T**: Opens Terminal (urxvt)__
**Mod1 + R**. Opens Dmenu__
**Mod1 + E**: Opens browser (Netsurf, in my setup's case)__
**Mod1 + C**: Closes Window.___
**Mod1 + M**: Exits DWM.__
