contains:

- Xresources file with palette & URxvt settings

# dwm Keybinds
**Mod1** is binded to **Alt**.
**Mod1 + T**: Opens Terminal (urxvt)
**Mod1 + R**. Opens Dmenu
**Mod1 + E**: Opens browser (Netsurf, in my setup's case)
**Mod1 + C**: Closes Window
**Mod1 + M**: Exits DWM.
